Generate web_root

There are two ways to generate plain HTML/CSS/JavaScript code from your dev_root.

The first way is to type 'ehc generate' (or 'ehc g' for short) from your commandline.
This wil automaticly parse all the dev_root projects.

The second avalible methode is the one tought four developing your website.
Using the build in webserver

This website is generated using Easy HTML Creator and Yard.
