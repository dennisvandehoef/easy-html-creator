This website contains the documentation for Easy HTML Creator.

Easy HTML creator is a Ruby Gem that maks it easy to genarate plain HTML websites.

The website can be programmed using HAML, Coffee and SASS if wished.
If SASS is used, it wil automatically include Bootstrap to your code.
